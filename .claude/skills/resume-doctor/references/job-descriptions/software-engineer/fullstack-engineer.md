## Role Summary

> **Full-Stack Software Engineer** — Designs, builds, and operates end-to-end features across the full stack (frontend, backend, infrastructure). Owns features from design through production operation. Collaborates with Product, Design, and Platform teams. Expected to own significant subsystems at Senior+ levels and define technical strategy at Staff+.

---

## Core Responsibilities (ATS-Keyword-Rich Bullet Bank)

### Feature Delivery (All Levels)
- Design, build, and ship **end-to-end features** across **React/TypeScript frontend** and **Node.js/Go/Python backend** services
- Develop **RESTful APIs** and **GraphQL APIs** with **OpenAPI/Swagger** specifications and **Schema Registry**
- Build **responsive, accessible UIs** using **React**, **TypeScript**, **Tailwind CSS**, **Next.js** (App Router, RSC, Server Actions)
- Implement **server-side rendering (SSR)**, **static site generation (SSG)**, **incremental static regeneration (ISR)**, **client-side rendering** patterns
- Write **unit tests (Vitest/Jest)**, **integration tests**, **E2E tests (Playwright/Cypress)** achieving >80% coverage
- Apply **feature flags** (LaunchDarkly, Unleash) for **progressive delivery**, **canary releases**, **A/B testing**

### Backend & Data (All Levels)
- Design and optimize **PostgreSQL** schemas: **indexing**, **partitioning**, **read replicas**, **connection pooling (PgBouncer)**
- Implement **Redis** caching layers: **read-through**, **write-through**, **write-behind**, **cache-aside**, **rate limiting**, **distributed locks**
- Build **event-driven architectures** using **Kafka**, **NATS**, **AWS EventBridge**, **Temporal** for **workflow orchestration**
- Implement **authentication/authorization**: **OAuth 2.0**, **OIDC**, **JWT**, **RBAC/ABAC**, **Session Management**, **MFA**
- Design **database migration strategies** (Prisma, Drizzle, raw SQL) with **zero-downtime deployments** (expand/contract pattern)

### Infrastructure & Operations (Senior+)
- Provision and manage **AWS infrastructure** using **Terraform**/**CloudFormation**/**CDK** (ECS/Fargate, Lambda, RDS, ElastiCache, CloudFront, S3, IAM)
- Containerize services with **Docker**; orchestrate with **Kubernetes (EKS/GKE/AKS)**, **Helm**, **Kustomize**, **ArgoCD/Flux**
- Implement **CI/CD pipelines** (GitHub Actions, GitLab CI) with **automated testing**, **staging**, **progressive delivery** (canary, blue-green)
- Establish **observability**: **OpenTelemetry** instrumentation, **Prometheus/Grafana** metrics, **Distributed Tracing (Jaeger/Tempo)**, **Structured Logging (JSON, Loki/ELK)**, **Error Tracking (Sentry)**, **Alerting (PagerDuty/Opsgenie)**
- Implement **feature flags**, **canary deployments**, **blue-green deployments**, **rollback automation**

### System Design & Architecture (Senior+)
- Design **scalable microservices architectures** with **Domain-Driven Design (DDD)**, **Event Sourcing**, **CQRS**, **Saga Pattern**
- Conduct **system design reviews**, **Architecture Decision Records (ADRs)**, **threat modeling** (STRIDE)
- Define **API contracts**, **service contracts**, **data contracts** with **Schema Registry** (Confluent, Apicurio)
- Lead **capacity planning**, **cost optimization**, **disaster recovery**, **chaos engineering** (Litmus, Gremlin)

### Technical Leadership (Staff+)
- Define **technical strategy** and **multi-year technical roadmap** aligned with business objectives
- Establish **engineering standards**: code style, architecture patterns, testing strategy, security baselines
- Lead **cross-team initiatives**: platform migrations, re-architecture, tech debt reduction, developer experience
- Mentor **Senior+ engineers**; grow **Staff+ engineers**; define **career ladders** and **hiring bars**
- Represent engineering in **executive reviews**, **board presentations**, **industry conferences**
