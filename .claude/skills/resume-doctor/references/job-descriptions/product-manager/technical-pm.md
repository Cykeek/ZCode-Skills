## Role Summary

> **Technical Product Manager** — Owns product strategy and execution for technical products (platforms, APIs, infrastructure, ML/AI, developer tools). Bridges business and engineering: translates customer needs into technical requirements, drives cross-functional delivery, measures impact via experimentation. At Senior+ defines platform strategy; at Staff+ sets company-wide technical product direction.

---

## Core Responsibilities (ATS-Keyword-Rich Bullet Bank)

### Product Strategy & Discovery (All Levels)
- Define **product vision** and **north star metric**; build **3-year strategy** using **Gibson/Borton framework**; allocate investment across **Core/Adjacent/Transformational** (70/20/10)
- Run **continuous discovery** (Teresa Torres): **Opportunity Solution Trees**, **JTBD interviews**, **usability tests**, **fake door tests**, **concept tests**; **10+ customer touches/week**
- Prioritize using **RICE/WSJF/ICE/Kano/Cost of Delay**; balance **new value**, **technical debt** (10-20% capacity), **risk reduction**; **quarterly planning** with **capacity-based roadmap**
- Own **outcome-oriented roadmap**: **Now/Next/Later**, **theme-based**, **date-confident for committed**, **date-agnostic for discovery**; map **dependencies**, **risks**, **mitigations**

### Technical Depth & Architecture (All Levels)
- Collaborate on **system design**: **distributed systems** (CAP, PACELC, consistency models), **microservices** (service mesh, API gateway, circuit breaker, retry, timeout, bulkhead), **event-driven** (Kafka, event sourcing, CQRS, saga)
- Design **APIs**: **REST** (resource modeling, HATEOAS, versioning, pagination, filtering), **GraphQL** (schema design, federation, subscriptions, N+1, complexity analysis), **gRPC** (Protobuf, unary/streaming, interceptors)
- Make **data decisions**: **SQL** (PostgreSQL, MySQL), **NoSQL** (DynamoDB, Cassandra, MongoDB, Redis), **OLAP** (ClickHouse, Snowflake, BigQuery); **modeling** (normalization, denormalization, partitioning, sharding)
- Define **infrastructure requirements**: **Cloud** (AWS/GCP/Azure), **Kubernetes** (EKS/GKE/AKS, operators, Helm, Kustomize), **Serverless** (Lambda, Cloud Run), **CI/CD** (GitHub Actions, ArgoCD, Spinnaker)

### Execution & Delivery (All Levels)
- Run **Agile ceremonies**: **Scrum** (sprint planning, daily, review, retro, refinement), **Kanban** (WIP limits, flow metrics, cycle time, throughput, CFD); **SAFe/LeSS** for scale (PI planning, ART)
- Write **PRDs**: **Problem**, **Solution**, **Metrics**, **Risks**, **Rollout**; **RFCs/ADRs** for architectural decisions; **acceptance criteria** (Gherkin/BDD); **Definition of Ready/Done**
- Manage **releases**: **canary**, **blue/green**, **feature flags** (LaunchDarkly, Unleash, GrowthBook); **rollback procedures**; **incident communication** (status page, stakeholder updates)
- Drive **cross-functional alignment**: **Engineering**, **Design**, **Data**, **Marketing**, **Sales**, **Support**, **Legal**, **Security**, **Finance**; **RACI**; **weekly syncs**, **monthly business reviews**

### Analytics & Experimentation (Mid+)
- Define **metric hierarchy**: **North Star**, **leading/lagging**, **input/output**, **guardrail**, **counter-metric**; **cohort retention**, **funnel conversion**, **LTV/CAC**, **payback**, **NRR/GRR**
- Design **experiments**: **A/B testing** (frequentist, Bayesian, sequential), **power analysis**, **sample size**, **MDE**, **p-hacking prevention**, **multiple comparisons** (Bonferroni, Benjamini-Hochberg), **holdouts**
- Analyze with **SQL** (CTEs, window functions, pivots, cohorts), **Python** (Pandas, NumPy, SciPy, Statsmodels), **R**; visualize in **Looker/Tableau/Mode/Superset/Metabase**
- Build **product analytics**: **event tracking** (Segment, RudderStack, Snowplow), **funnel**, **retention**, **path analysis**, **attribution**, **feature adoption**, **power user curve**

### Platform / Developer Experience / AI PM (Senior+)
- Own **platform product**: **Internal Developer Platform** (Backstage, Platform Engineering), **API Platform** (Kong, Apigee, AWS API Gateway), **Data Platform**, **ML Platform**, **Infrastructure Platform**
- Optimize **Developer Experience**: **SDK design** (TypeScript, Python, Go, Java), **documentation** (OpenAPI, Redoc, Mintlify), **onboarding** (Time to First Hello World), **DX metrics** (NPS, adoption, time-to-value)
- Lead **AI/ML products**: **Model lifecycle** (training → evaluation → deployment → monitoring), **RAG/Fine-tuning/RLHF**, **evaluation** (offline/online, LLM-as-judge, RAGAS), **guardrails**, **cost/token management**
- Integrate **MLOps**: **Feature Store**, **Model Registry**, **Experiment Tracking**, **CI/CD for ML**, **Drift Detection**, **Model A/B Testing**, **Shadow/Canary Deployment**

### Strategy & Leadership (Staff+)
- Define **company-wide technical product strategy**: **multi-year roadmap**, **build vs. buy**, **vendor evaluation**, **platform consolidation**, **technical leverage**
- Build **platform serving 500+ engineers**: **self-serve**, **golden paths**, **paved roads**, **adoption >80%**, **developer NPS >50**, **time-to-prod <1 day**
- Lead **high-leverage products**: **API platform** (1B+ calls/day), **ML platform** (10K+ models), **Data platform** (100TB/day), **Developer platform** (500+ devs)
- Grow **PM org**: **hiring bar**, **career ladder**, **mentorship**, **PM guild**, **community of practice**; **industry speaking**, **thought leadership**
