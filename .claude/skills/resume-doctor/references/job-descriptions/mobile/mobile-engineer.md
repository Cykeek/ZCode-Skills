## Role Summary

> **Mobile Engineer** â€” Builds, optimizes, and ships native (iOS/Android) and cross-platform (React Native, Flutter, KMP) mobile applications. Owns end-to-end mobile lifecycle: architecture, performance, CI/CD, release management, testing, platform features, security, analytics. At Senior+ level: defines mobile technical strategy, builds platform infrastructure, drives cross-platform unification, influences product direction. At Staff/Principal: sets org-wide mobile vision, represents mobile to executive leadership, leads transformations, crisis management.

## Core Responsibilities (ATS-Keyword-Rich Bullet Bank)

### Native iOS Development (All Levels)
- Develop **iOS applications** using **Swift**, **SwiftUI**, **UIKit**, **Combine**, **Core Data**, **Core Animation**, **Metal**, **ARKit**, **CoreML**
- Implement **modern iOS patterns**: Swift Concurrency (async/await), Structured Concurrency, Actors, Swift Testing, SwiftData, Observation
- Optimize **iOS performance**: App Startup (pre-main, dyld), 120fps ProMotion scrolling, memory (ARC, retain cycles, Instruments), battery (Location, Background Tasks)
- Leverage **platform capabilities**: Widgets, Live Activities, App Intents, Shortcuts, App Clips, WatchOS, tvOS, VisionOS, HealthKit, Wallet, MapKit
- Manage **iOS release pipeline**: Xcode Cloud / Fastlane / Bitrise, TestFlight (Internal/External), App Store Connect, phased rollout, certificates/profiles (match)

### Native Android Development (All Levels)
- Develop **Android applications** using **Kotlin**, **Jetpack Compose**, **View System**, **Coroutines**, **Flow**, **Room**, **DataStore**, **CameraX**, **ML Kit**
- Implement **modern Android patterns**: Compose (Declarative UI), Kotlin Flows, State Hoisting, Unidirectional Data Flow, Hilt/Koin DI, Baseline Profiles
- Optimize **Android performance**: Startup (Baseline Profiles, App Startup lib), 60/90/120fps (JankStats, Perfetto), memory (LeakCanary, GC), battery (WorkManager, Doze)
- Leverage **platform capabilities**: Widgets, Wear OS, Android Auto, TV, Instant Apps, App Shortcuts, Bubbles, Picture-in-Picture, Health Connect, Credential Manager
- Manage **Android release pipeline**: Gradle (KTS, version catalogs), Play Console (Internal/Closed/Open/Production), staged rollout, App Bundle, Play Integrity API

### Cross-Platform Development (Mid+)
- **React Native**: TypeScript, Hermes, Fabric, TurboModules, CodePush, Expo (EAS), Reanimated, Gesture Handler, Skia, Turbo Native Modules
- **Flutter**: Dart, Impeller, Skia, Shorebird (code push), Flutter Web/desktop, Platform Channels, FFI, Dart Frog, Riverpod/Bloc/Provider
- **Kotlin Multiplatform (KMP)**: expect/actual, Compose Multiplatform, Shared Business Logic, SQLDelight, Ktor, Koin, Gradle Plugin, Xcode integration
- Evaluate **trade-offs**: shared code % (target >60%), platform-specific optimization, binary size, debuggability, team expertise, hiring pool

### Mobile Architecture & Modularization (Senior+)
- Design **mobile architecture**: Clean Architecture, MVVM, MVI, VIPER, Unidirectional Data Flow, Coordinator, Router, Feature-First organization
- Implement **modularization**: Dynamic frameworks (iOS), Gradle modules (Android), SPM packages, Tuist, XcodeGen, Bazel, Buck, Monorepo strategies
- Define **state management**: Combine, RxSwift, RxJava, Coroutines/Flow, Redux, TCA, Bloc, Cubit, Riverpod, MobX, Provider, Signals
- Establish **dependency injection**: Swift DI, Hilt, Koin, Dagger, Swinject, Needle, Factory, Manual DI, Property Wrappers, Macros
- Build **shared infrastructure**: Networking layer (Apollo, GraphQL, gRPC, REST), Database abstraction, Analytics wrapper, Feature flags, Logging

### Performance Engineering (Senior+)
- **Startup Optimization**: Cold/warm/hot start, pre-main time, dyld optimization, class loading, Baseline Profiles, App Startup library, lazy initialization
- **Rendering Performance**: 60/120fps, frame drops (JankStats), overdraw, GPU profiling (Metal/Compose), layout inspection, recomposition optimization
- **Memory Management**: Leaks (LeakCanary, Malloc Stack Logging, Instruments), retain cycles (weak/unowned), ARC/GC tuning, image caching, pagination
- **Battery Optimization**: Location batching (FusedLocationProvider), network batching, WorkManager/BackgroundTasks, Doze/App Standby, Foreground Services
- **Network Optimization**: HTTP/2/3, QUIC, connection pooling, caching (Cache-Control, ETag), offline-first (WatermelonDB, Realm, Room), GraphQL normalization

### CI/CD & Release Management (All Levels)
- Build **mobile CI/CD**: Fastlane (match, snapshot, gym, pilot, deliver, supply), Bitrise, GitHub Actions, GitLab CI, CircleCI, Xcode Cloud, Jenkins
- Automate **distribution**: TestFlight, App Store Connect API, Google Play Console API, Firebase App Distribution, App Center, Internal Testing tracks
- Implement **release automation**: Versioning (semver, build numbers), Changelogs, Screenshots (Fastlane snapshot), Metadata, Certificates/Profiles
- Enable **code push**: CodePush (React Native), Shorebird (Flutter), Expo Updates, instant updates without App Store review
- Monitor **release health**: Crashlytics/Sentry/Bugsnag adoption tracking, phased rollout monitoring, rollback automation, emergency hotfix process

### Testing Strategy (All Levels)

- **UI Testing**: XCUITest, Espresso, Detox, Appium, Maestro, EarlGrey, KIF, Snapshot testing (iOSSnapshotTestCase, Shot, Paparazzi, Golden Tests)

### Platform Features & Integrations (Mid+)
