## Role Summary

> **UI Designer** — Crafts pixel-perfect, accessible, and brand-aligned interfaces. Translates wireframes and UX flows into high-fidelity, responsive designs. Builds and governs design systems. Partners with UX, Engineering, and Brand to deliver cohesive visual experiences across platforms. At Senior+ levels, defines visual language, leads design system strategy, and elevates organizational craft standards.

---

## Core Responsibilities (ATS-Keyword-Rich Bullet Bank)

### Visual & UI Design (All Levels)
- Create **high-fidelity UI mockups** for **web**, **mobile (iOS/Android)**, **desktop**, **tablet** using **Figma** with **Auto Layout**, **Variants**, **Component Properties**, **Design Tokens**
- Design **responsive layouts** across **breakpoints** (Mobile: 320-480px, Tablet: 768-1024px, Desktop: 1440px+, Wide: 1920px+) with **fluid typography**, **flexible grids**, **container queries**
- Define **component states**: **Default**, **Hover**, **Focus**, **Active**, **Disabled**, **Loading**, **Error**, **Empty**, **Success** for every interactive element
- Apply **visual hierarchy** through **typography scale**, **color semantics**, **spacing system**, **elevation/shadow system**, **border radius system**
- Create and maintain **iconography system** (system icons, custom icons, **SVG optimization**, **icon font / SVG sprite**)
- Design **data visualizations**: charts, graphs, dashboards, tables with **accessible color palettes**, **patterns**, **labels**

### Design Systems & Component Library (Mid+)
- Architect **design token system**: **Color** (primitive + semantic), **Spacing**, **Typography**, **Shadows**, **Border Radius**, **Motion**, **Z-Index**, **Breakpoints**
- Build **component library** in **Figma**: foundations → atoms → molecules → organisms → templates → pages (Atomic Design)
- Publish and version **Figma Libraries**; manage **library updates**, **migration guides**, **breaking change communication**
- Document **component anatomy**, **usage guidelines**, **accessibility requirements**, **implementation specs** (Zeroheight, Notion, Storybook Docs)
- Ensure **cross-platform parity**: **Web (CSS Variables)**, **iOS (SwiftUI/UIKit)**, **Android (Jetpack Compose/XML)**, **React Native**

### Motion & Interaction Design (Mid+)
- Design **micro-interactions**: button press, toggle, dropdown, tooltip, toast, loader, skeleton, page transitions
- Specify **motion tokens**: **duration**, **easing**, **delay**, **stagger** for consistent feel
- Prototype in **Figma** (Smart Animate, Interactive Components) or **Framer/ProtoPie/Principle** for complex flows
- Collaborate with engineers on **implementation feasibility**, **performance budgets**, **reduced motion** compliance

### Engineering Handoff & Design QA (All Levels)
- Prepare **design specs** in **Figma DevMode**: spacing, typography, color, shadows, breakpoints, states, interactions
- Export **production-ready assets**: **SVG** (icons, illustrations), **PNG/WebP** (raster), **PDF** (prints)
- Conduct **visual QA** / **design review** in **staging**: pixel accuracy, responsive behavior, motion fidelity, accessibility
- Partner with engineers on **Storybook** component development, **Chromatic** visual regression testing

### Brand & Marketing Collaboration (Senior+)
- Translate **brand guidelines** into **product UI language**: color palette adaptation, typography pairing, imagery style
- Collaborate with **Brand/Marketing Design** on **landing pages**, **campaign assets**, **email templates**, **social templates**
- Evolve **design language** with product maturity: from MVP → scale → platform

### Strategic Leadership (Senior+)
- Define **visual design principles**, **quality bar**, **review criteria** for org
- Lead **design system strategy**: token architecture, component roadmap, adoption metrics, governance model
- Mentor designers: **portfolio reviews**, **craft critiques**, **skill development**, **career progression**
- Drive **accessibility visual compliance** org-wide: contrast audits, focus indicator standards, motion reduction
